## Структура папки

```
lessons/<проект>/
├── README.md          — описание проекта, программа, обратная связь Юли
├── lesson-01.md       — урок 1
├── lesson-02.md       — урок 2
└── ...
```

## Правила
1. Каждый урок — отдельный файл `lesson-XX.md`
2. `README.md` проекта содержит: программу, прогресс, обратную связь Юли
3. При отправке урока — сохранять в файл и коммитить
4. Обратную связь Юли дописывать в README проекта
