# 🍲 Кухня с душой + Вкусные субботы

> Кулинарные рассылки Эмбер для Юли
> Расписание: **Среда** (вдохновение на конец недели) + **Воскресенье** (вдохновение на будни)
> Начало: март 2026

## Формат рассылки

### Среда — «Навстречу выходным»
1. 🍽 Идея пятничного ужина (что-то особенное, с запасом на сб-вс)
2. 🎯 **Кулинарный квест недели** — блюдо из кафе, которое повторяем дома
   - Описание блюда + пошаговая инструкция + ссылки на рецепты
   - Из файла `family-kitchen.md` → раздел «Кулинарные квесты»
3. 🛒 Что докупить (короткий список под квест + ужин)
4. 👧 Идея для Вкусной субботы (с девочками)

### Воскресенье — «Новая неделя»
1. 🌅 Завтраки на неделю (3 идеи, простые)
2. 🍲 Обед-герой недели (одно блюдо на несколько дней)
3. 🎯 **Итог квеста** — получилось? Что изменить? Фото?
4. 💡 Лайфхак недели (хранение, заготовка, экономия)
