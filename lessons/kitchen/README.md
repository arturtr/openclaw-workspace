## Формат рассылок

### Среда — «Вкусные субботы» 🍕
> Контент к выходным: что готовить с детьми + квесты для взрослых

1. 👧 Идея для субботней готовки с девочками (рецепт + список покупок)
2. 🍽 Идея пятничного ужина (что-то особенное)
3. 🎯 **Квест для Юли** — блюдо, которое она хочет освоить
4. 🎯 **Квест для Артура** — блюдо для него (мужское, интересное)
5. 🎯 **Квест на двоих** — приготовить вместе вечером (Sacred Couple на кухне)
6. 🛒 Общий список покупок

### Воскресенье — «Кухня с душой» 🍲
> Планирование меню на неделю + итоги квестов

1. 🌅 Завтраки на неделю (3 идеи)
2. 🍲 Обед-герой (одно блюдо на несколько дней)
3. 🥗 Ужины (лёгкие, быстрые)
4. 📝 **Итог квестов** — получилось? Что изменить? Фото?
5. 🎯 **Новый квест на неделю** (чередование: Юля → Артур → на двоих)
6. 💡 Лайфхак недели
7. 🛒 Список покупок под план
