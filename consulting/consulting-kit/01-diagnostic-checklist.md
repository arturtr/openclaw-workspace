# Диагностический чеклист: первая встреча с клиентом

**Цель**: структурированное интервью на 30-45 минут. Результат — понимание текущего состояния данных в компании и определение точки входа для помощи.

**Как использовать**: распечатать или открыть на планшете. Задавать вопросы, отмечать ответы. После встречи — заполнить оценку зрелости (документ 02).

---

## Блок 1: Бизнес-контекст (5 мин)

> Цель: понять масштаб бизнеса и что для собственника важно.

- [ ] **Чем занимается компания?** (отрасль, основной продукт/услуга)
- [ ] **Сколько сотрудников?** ___
- [ ] **Годовая выручка (порядок)?** до 10М / 10-50М / 50-200М / 200М+
- [ ] **Сколько точек продаж / филиалов?** ___
- [ ] **Какую главную задачу бизнеса решаете сейчас?** (рост, оптимизация, выход на новый рынок)
- [ ] **Что вас беспокоит в данных прямо сейчас?** (свободный ответ)

## Блок 2: Текущие инструменты (5 мин)

> Цель: понять технический ландшафт.

- [ ] **Где ведёте учёт?** 1С / Excel / Google Sheets / облачная бухгалтерия / другое
- [ ] **Есть ли CRM?** Нет / Bitrix24 / amoCRM / другое: ___
- [ ] **Где хранятся данные о продажах?** 1С / Excel / CRM / нигде
- [ ] **Есть ли сайт / интернет-магазин?** Нет / Tilda / Bitrix / 1С-Битрикс / другое
- [ ] **Используете ли аналитику сайта?** Нет / Яндекс.Метрика / Google Analytics
- [ ] **Есть ли складской учёт?** Нет / 1С / МойСклад / Excel
- [ ] **Сколько разных систем используете?** (список)

## Блок 3: Как принимаются решения (10 мин)

> Цель: понять текущий процесс — от вопроса до ответа.

- [ ] **Когда нужна цифра (выручка за месяц, остатки на складе) — как её получаете?**
  - Звоню/пишу бухгалтеру
  - Смотрю в 1С сам
  - Открываю Excel-файл
  - Смотрю дашборд
  - Не знаю, как получить

- [ ] **Сколько времени уходит на получение ответа?**
  - Мгновенно (дашборд)
  - Минуты (открыл файл, нашёл)
  - Часы (нужно попросить кого-то собрать)
  - Дни (нужно ждать отчёт)
  - Не получаю вообще

- [ ] **Кто готовит отчёты?** (должность, сколько человек)
- [ ] **Как часто готовятся отчёты?** Ежедневно / Еженедельно / Ежемесячно / По запросу
- [ ] **Есть ли отчёты, которые вы хотели бы видеть, но не видите?** (свободный ответ)
- [ ] **Были ли случаи, когда приняли неверное решение из-за отсутствия данных?** (пример)

## Блок 4: Боли и потери (10 мин)

> Цель: найти где данные стоят денег. Это основа для ROI-обоснования.

- [ ] **Сколько часов в неделю сотрудники тратят на ручной сбор данных?** ___ч
- [ ] **Бывают ли расхождения между отчётами разных отделов?** Часто / Иногда / Нет
- [ ] **Были ли случаи пересорта / недостачи на складе?** Да, часто / Иногда / Нет
  _(пересорт — когда на складе лежит не тот товар, который числится в учёте)_
- [ ] **Можете ли вы прямо сейчас сказать, какой товар самый прибыльный?** Да / Нет / Примерно
- [ ] **Знаете ли точную себестоимость каждого продукта/услуги?** Да / Нет / Примерно
- [ ] **Есть ли у вас план/факт по ключевым метрикам?** Да / Нет / Частично
- [ ] **Теряете ли клиентов, которых можно было бы удержать при наличии данных?** (пример)

## Блок 5: IT-ресурсы (5 мин)

> Цель: понять кто будет поддерживать решение после внедрения.

- [ ] **Есть ли IT-отдел / IT-специалист?** Да (___ чел) / Нет / Аутсорс
- [ ] **Кто-то пишет формулы в Excel / макросы / скрипты?** Да (кто: ___) / Нет
- [ ] **Есть ли свой сервер?** Да / Нет / Облако (какое: ___)
- [ ] **Бюджет на IT-инструменты в месяц?** 0 / до 10k / 10-50k / 50k+
- [ ] **Готовы ли обучить сотрудника работе с новым инструментом?** Да / Нет / Зависит от инструмента

## После встречи: быстрая оценка

### Уровень зрелости данных (отметить)

| Уровень | Описание | Признак |
|---------|----------|---------|
| 1. Хаос | Данные разрозненны, решения на интуиции | Нет единого учёта, Excel-файлы не связаны |
| 2. Учёт | Есть 1С/CRM, но данные не анализируются | Данные собираются, но отчёты ручные |
| 3. Отчётность | Регулярные отчёты, но ручные | Есть ответственный за отчёты, но всё в Excel |
| 4. Аналитика | Дашборды, автоматические отчёты | BI-инструмент, данные обновляются сами |
| 5. Data-driven | Решения основаны на данных, есть прогнозы | A/B тесты, предиктивная аналитика |
