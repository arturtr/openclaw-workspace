# Гипнотерапия. Практическое руководство

- **Авторы:** Гельмут Карл, Дженнифер Бойз
- **Оригинал:** *Hypnotherapy: A Practical Handbook*
- **Категория:** 🧠 Гипнотерапия, психотерапия
- **Статус:** 📦 есть
- **Страниц:** ~350

## Описание

Практическое руководство по гипнотерапии для начинающих (и справочник для опытных). Все программы получили практическое применение, каждый случай подробно описан, приведены сценарии сеансов гипнотерапии.

## Ключевые темы

- Гипнотерапия: теория и практика
- Сценарии сеансов гипнотерапии
- Техники наведения транса
- Работа с подсознанием
- Практические случаи с описаниями

## 🗝️ Почему стоит открыть эту книгу

Гипнотерапия — корень эриксоновского коучинга. Аткинсон (#75, #76) — последовательница Милтона Эриксона, который был гипнотерапевтом. Эта книга — понимание КОРНЕЙ того подхода, который стал трансформационным коучингом.

## 💎 Сокровища внутри

- Сценарии сеансов — готовые скрипты
- Техники наведения транса — основа «разговорного гипноза»
- Подробные описания случаев — учись на практике
- Связь гипнотерапии и коучинга — через Эриксона
- Глаз на обложке не случаен — работа с вниманием и фокусом
