# Сказки для коучинга

- **Автор:** Маргарет Паркин
- **Подзаголовок:** Как использовать сказки и метафоры в работе с отдельными людьми и рабочими группами
- **Оригинал:** *Tales for Coaching*
- **Серия:** Сказки для менеджеров
- **Категория:** 🧠 Коучинг, метафоры
- **Статус:** 📦 есть
- **ISBN:** 978-5-98124-462-9
- **Издательство:** Добрая книга, 2009
- **Страниц:** 304

## Описание

50 сказок и историй для использования в коучинге и работе с малыми группами. Каждая история снабжена пояснением: в какой ситуации использовать, какие вопросы задать, на что обратить внимание. Сказочники существовали всегда — книга учит использовать древнее искусство историй в современной практике.

## Ключевые темы

- 50 готовых сказок/историй для коуч-сессий
- Метафоры как инструмент трансформации
- Работа с индивидуальными клиентами через истории
- Работа с малыми группами
- Техника сторителлинга в коучинге
- Подбор сказки под ситуацию клиента

## Для кого полезна

- Юля: прямой инструмент для практики — готовые истории для сессий. Перекликается с «делать сокрытое видимым» через метафору. Можно использовать в Eva Divina и индивидуальной работе. Связка с МАК (Ингерлейб) — образы + истории = мощная комбинация
