# Cinelogia для молодёжи (Синемалогия для молодёжи)

- **Автор:** Антонио Менегетти (Antonio Meneghetti)
- **Подзаголовок:** Кино. Онтопсихологический подход
- **Категория:** 🧠 Психология, кино
- **Статус:** 📦 есть
- **ISBN:** 978-5-906601-09-4

## Описание

Онтопсихологическая синемалогия — метод использования кинофильмов как инструмента психологического и социологического анализа. Кино как зеркало бессознательных процессов. Адаптация метода для молодёжи.

## Ключевые темы

- Синемалогия: анализ фильмов через онтопсихологию
- Кино как проекция бессознательного
- Образы и символы в кинематографе
- Самопознание через просмотр фильмов
- Онтопсихологический подход Менегетти

## Для кого полезна

- Юля: инструмент для работы с образами и символами, пересечение с «делать сокрытое видимым», возможный формат для клуба (синемалогические встречи)
