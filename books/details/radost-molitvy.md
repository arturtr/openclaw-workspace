# Радость молитвы. Благодать ежедневного общения с Богом

**Составитель:** Г.А. Сулаженко
**Категория:** 🙏 Вера, духовность
**Статус:** 📦
**ISBN:** 978-5-904737-99-3
**Издательство, год:** Виссон (СПб), 2012 (3-е изд.)
**Страниц:** 416

## ⚠️ Может вводить в заблуждение
Причина: Издательство «Виссон» = протестантское. Тексты Священного Писания приводятся по Синодальному переводу Библии. Сборник молитв, каждая из которых может стать уроком для тех, кто ежедневно предстоит перед Богом. 366 уроков — хорошая духовная школа. Но богословские комментарии — в евангельской традиции.

## Описание
366 молитв на каждый день года (включая 29 февраля). Сборник для ежедневного предстояния перед Богом. Составитель Г.А. Сулаженко. «Эта книга будет замечательным подарком для ваших друзей и близких».

## Ключевые темы
- 366 молитв = по одной на каждый день
- Библейская основа каждой молитвы
- Ежедневное общение с Богом как практика
- Школа молитвы для начинающих и продвигающихся

## 🗝️ Почему стоит открыть эту книгу
416 страниц = 366 дней молитвы. Формат «одна молитва в день» — просто, доступно, регулярно. Перекличка с #271 (Чепмен, молитвенная минутка) — оба протестантские, оба ежедневные, но этот — чистая молитва без семейного контекста.

## 💎 Сокровища внутри
- 366 готовых молитв — не нужно формулировать самому
- Синодальный перевод — совпадает с православным
- Формат: утром открыл → помолился → закрыл

## ⏱️ Время чтения
По 1 молитве в день = 366 дней

## Для кого полезна
**Семья** — для ежедневной молитвенной практики. Дополняет православный молитвослов (#121).
