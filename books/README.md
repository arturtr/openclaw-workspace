# Семейная библиотека — правила каталогизации

## Где хранится библиотека

**Obsidian vault:** `/home/artur/Dropbox/obsidian-vault/31 - Библиотека/`

Каждая книга = отдельный файл в формате Obsidian с YAML frontmatter.

**Старый каталог** (`catalog.md`, `details/`) — архив. Новые книги сюда НЕ добавлять.

## Формат файла книги (ОБЯЗАТЕЛЬНО)

Имя файла: `Название книги — Автор.md` (на русском, читаемое)

Если название длиннее 120 символов — обрезать по последнему слову.

### YAML frontmatter

```yaml
---
title: "Полное название книги"
author: "Автор"
date: ГГГГ-ММ-ДД
status: 📦
pages: 
isbn: ""
publisher: ""
owner: ""
tags:
  - тема/ТЕМА
  - тип/книга
  - статус/сохранено
source: ""
related: []
---
```

### Теги — маппинг категорий

| Категория | Тег |
|-----------|-----|
| Вера, духовность | `тема/вера` |
| Дом, семья, воспитание | `тема/домоводство` |
| Бизнес, маркетинг, продуктивность | `тема/бизнес` |
| Психология, саморазвитие | `тема/психология` |
| Коучинг, менторинг | `тема/коучинг` |
| Чай, культура | `тема/чай` |
| Данные, аналитика, IT | `тема/данные` |
| Детские | `тема/дети` |
| Художественная литература | `тема/литература` |
| Наука, история | `тема/наука` |
| Священное в повседневном | `тема/homo-divinus` |

Книга может иметь 2-3 тематических тега. Тип всегда `тип/книга`.

### Статусы

- 📖 читаю
- ✅ прочитал
- 📌 хочу
- 📦 есть (не читали)

### Тело файла

```markdown
# Название книги

**Автор:** Имя
**ISBN:** 
**Издательство, год:** 
**Страниц:** 

## Описание
О чём книга (2-3 предложения)

## Ключевые темы
- Тема 1
- Тема 2

## Почему стоит открыть эту книгу
Зажечь любопытство (2-3 предложения)

## Сокровища внутри
- 3-5 конкретных жемчужин (техники, идеи, инсайты)

## Время чтения
X стр. → 5 стр/день = Y дней | 10 стр/день = Z дней | 20 стр/день = W дней

## Для кого полезна
Кому из семьи и в какой ситуации
```

## Обязательное правило

**Файл в `31 - Библиотека/` создаётся СРАЗУ при добавлении книги. Не бывает книги без файла.**

## Маркировка: может вводить в заблуждение

Если книга не вписывается в православную традицию и может содержать псевдохристианские, эзотерические или сектантские идеи:

В файле книги добавить блок:
```
## Может вводить в заблуждение
Причина: [конкретное объяснение]
```

Цель: не цензура, а информированный выбор.

## Детские книги — дополнительно

Книги выбирают РОДИТЕЛИ (Юля/Артур), не дети.
Детская книга также содержит:
- **Возраст:** Маша 8 / Алёна 6 / обе
- **Ситуация:** когда дать почитать
- **Чему учит:** главный посыл
- **Родителю:** заметки
