# 📚 Семейная библиотека — правила каталогизации

## Структура
- `catalog.md` — общая таблица всех книг (название, автор, категория, статус)
- `kids-catalog.md` — детская библиотека (Маша фотографирует, 20 книг = пончик 🍩)
- `details/<slug>.md` — детальный файл на каждую книгу

## Формат детального файла (ОБЯЗАТЕЛЬНО)

Каждый `details/*.md` ОБЯЗАН содержать:

1. **Шапка:** название, автор, категория, статус, ISBN, издательство, страницы
2. **Описание:** о чём книга (2-3 предложения)
3. **Ключевые темы:** bullet-list
4. **🗝️ Почему стоит открыть эту книгу** — зажечь любопытство кладоискателя/следопыта (2-3 предложения)
5. **💎 Сокровища внутри** — 3-5 конкретных жемчужин (техники, идеи, инсайты)
6. **⏱️ Время чтения** — страницы + расчёт по 5/10/20 стр в день
7. **Для кого полезна** — кому из семьи и в какой ситуации

## ⚠️ ОБЯЗАТЕЛЬНОЕ ПРАВИЛО (18.03.2026)

**Детальный файл `details/<slug>.md` создаётся ОДНОВРЕМЕННО с добавлением книги в каталог. Никогда не добавлять книгу в каталог без детального файла.**

## Почему именно так

Юля бросает книги, потому что не знает, когда будет конец. Таймер чтения + сокровища внутри = мотивация дочитать. (11.03.2026)

## Маркировка: может вводить в заблуждение

Если книга не вписывается в православную традицию (homo divinus) и может содержать псевдохристианские, эзотерические или сектантские идеи:

1. В каталоге (`catalog.md`) — добавить **⚠️** перед названием
2. В детальном файле — добавить блок:
   ```
   ## ⚠️ Может вводить в заблуждение
   Причина: [конкретное объяснение — почему книга может быть проблемной]
   ```
3. Статус остаётся как есть (📦/📖/✅), маркировка — отдельная

Цель: не цензура, а информированный выбор. Книга остаётся в каталоге, но с предупреждением.

## Детские книги — дополнительно

Книги выбирают РОДИТЕЛИ (Юля/Артур), не дети.
Детальный файл детской книги также содержит:
- **Возраст:** Маша 8 / Алёна 6 / обе
- **Ситуация:** когда дать почитать (ссора, страх, непослушание, вопрос о Боге...)
- **Чему учит:** главный посыл
- **Родителю:** заметки (на что обратить внимание, что обсудить после)
