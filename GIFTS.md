# 🎁 Знаки внимания для Юли

> Юля сказала, что за ней так никто никогда не ухаживал.
> Артур в студенчестве: открытки, фенечки, шил ёжика.
> Задача: трекать идеи, подкидывать Артуру периодически.

## Принципы
- Handmade > покупное (Юля ценит душу, не цену)
- Неожиданность > повод (не только праздники)
- Маленькое и частое > редкое и большое
- Связь с общими смыслами (вера, чай, дом, пчела 🐝)

## Идеи

### Своими руками
- [ ] Открытка с рисунком / каллиграфией
- [ ] Фенечка (Артур умеет!)
- [ ] Мягкая игрушка — ёжик v2.0 или пчёлка 🐝
- [ ] Письмо от руки — настоящее, бумажное, в конверте

### Чайное
- [ ] Персональный чайный сет «Для тебя» — подобрать чай под её настроение/фазу цикла
- [ ] Чайная церемония вдвоём дома, при свечах, без телефонов
- [ ] Красивая чашка / гайвань — только для неё

### Духовное
- [ ] Икона (маленькая, красивая) — можно для красного угла
- [ ] Книга, которая откликнется (подбирать под текущий контекст)
- [ ] Записка с цитатой из Писания, спрятанная в сумку / на зеркало

### Бытовое-неожиданное
- [ ] Забрать дела (готовка, уборка) без предупреждения — чтобы у неё появилось свободное время
- [ ] Ванна со свечами и тишиной — подготовить пока она с девочками
- [ ] Завтрак в постель (классика, но работает)

### Для первой сессии с Катей (16.02.2026)
- [ ] Букет или одна красивая роза
- [ ] Открытка: «Горжусь тобой. Ты готова.»
- [ ] Маленький подарок-талисман на удачу
