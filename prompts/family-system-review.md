# Family System Review — Coaching Session Prompt

**Purpose**: Провести ревизию семейной системы openclaw-workspace через coaching-сессию. Формат: one-on-one, один вопрос за раз, как наставник мирового уровня.

**Context for Claude**: Artur провёл такую же ревизию своего карьерного плана (jiffy-data-docs). Ключевые открытия:
- Situated learning > формальные расписания
- Frameworks + direction > жёсткие процедуры
- Process > result — создать рамки, дать жизни жить внутри них
- Change Agency — быть драйвером без формальной власти
- North Star важнее milestone'ов

Artur хочет перенести эти принципы в семейную систему.

---

## Before Starting

1. Read the entire workspace to understand current state:
   - `CLAUDE.md` — system instructions
   - `FAMILY.md`, `IDENTITY.md`, `SOUL.md`, `HEARTBEAT.md` — philosophy and values
   - `ARTUR.md`, `YULIA.md` — individual profiles
   - `*-DEVELOPMENT.md` — development plans for each family member
   - `AGENTS.md`, `TOOLS.md` — current infrastructure
   - `KANBAN.md` — task management
   - `MEMORY.md`, `memory/` — accumulated knowledge
   - `CURRENT_CONTEXT.md` — latest state
   - `agents/` — agent configurations
   - `family/` — family-specific docs

2. After reading, summarize what you understood in 10 lines. Ask Artur to confirm or correct.

---

## Session 1: Artur solo

Interview format. One question at a time. Explore:

1. **Current state**: What works in the system right now? What doesn't?
2. **Agents**: Кузя, Эмбер, Марк, Лука — who does what? What's useful, what's noise?
3. **Yulia's experience**: Does the system help her or create friction?
4. **Values**: What do Artur and Yulia want from family life? Not goals — values.
5. **North Star**: If this system worked perfectly — what would a typical week feel like?
6. **Frustrations**: What breaks, what's chaotic, what needs fixing first?
7. **Philosophy**: What principles should guide the system? (Connect to work principles: clarity, frameworks, let life happen)

After the interview:
- Summarize findings
- Propose structure changes
- Draft a family philosophy (like work philosophy, but for home)
- Update relevant files

---

## Session 2: Artur + Yulia together

Same format, but both participate. Key additions:

1. Share Artur's solo findings — does Yulia agree?
2. **Yulia's perspective**: Same questions, her answers
3. **Alignment**: Where do values match? Where do they differ?
4. **Requests**: What does each person need from the system?
5. **Agents**: Which agents help Yulia? What's missing?
6. **Joint North Star**: Shared vision for the family system

After the session:
- Merge both perspectives
- Build system architecture from actual needs
- Set direction (not schedule) for the system

---

## Principles for the review

- One question at a time
- No judgment — the current chaos is valid, it brought them here
- Frameworks > procedures
- Direction > schedule
- Both voices matter equally
- The system serves the family, not the other way around
- Connect to what already works, don't rebuild from scratch
