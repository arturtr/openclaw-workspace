# Модель жизни исследователя HD (по Теренсу Тао + адаптация для Юли)

> Дата: 25.03.2026
> Источник: https://habr.com/ru/articles/1013942/ (обзор интервью Тао, 20.03.2026)
> Контекст: Юля хочет «жить как исследователь homo divinus — и писать глубочайшие статьи высокого уровня»

---

## Ключевые тезисы Тао

1. **Идеи стоят ноль.** AI генерирует тысячи гипотез. Ценность сместилась на верификацию и отбор
2. **Шире, но не глубже.** Вспомогательное ускорилось в 5 раз. Ядро — на ноль. Ручка и бумага
3. **Хитрость ≠ интеллект.** AI прыгает, но не карабкается. Нет кумулятивного процесса
4. **Серендипити под угрозой.** Случайные находки исчезают. Нужно создавать условия для них
5. **Генерация без верификации = слоп.** Кеплер без Тихо Браге = мусор

## Адаптация для HD

**Юля = Тихо Браге, не Кеплер.**
Суперсила — не «придумывать», а проживать, наблюдать, верифицировать опытом.
AI не может жить. Юля — может. Это и есть ценность.

## Формула исследователя HD

> **Читаю → проживаю → замечаю → записываю → делюсь**

### Ритм дня:
- 🌅 Утро: чтение (30-60 мин). Один источник. Не «изучить» — прожить
- 📝 После чтения: одна запись. Не конспект — «что это меняет в моей жизни?»
- 🌿 День: одно дело руками (керамика, фото, кухня) = серендипити через тело
- 🌙 Вечер: одна запись в блог. Из утреннего чтения + дневного проживания

### Роли:
- **Юля:** ядро (чтение, проживание, молитва, текст, фото, сессии). Глубина = только из неё
- **Эмбер:** вспомогательное (исследования, переводы, подборки, оформление, обратная связь, наставники)
- **Марк:** визуал, структура, сайт

### Серендипити:
- Книжные квесты = случайные находки на своей полке
- Фотопрактика = замечать через камеру
- Прогулки, керамика, кухня = тело как источник инсайтов

### Статьи высокого уровня:
Рождаются не из исследования, а из **проживания + осмысления.**
Пост про Льюиса и Брак — пример: из утра, из боли, из правды → текст уровня манифеста.
Продолжать так: не «писать статьи», а «жить и записывать, что увидела».

## Как Эмбер помогает

| Этап | Юля | Эмбер |
|------|-----|-------|
| Читаю | Выбирает + читает | Квесты с полки, подборки, выжимки |
| Проживаю | Живёт | Напоминает: «что заметила?» |
| Замечаю | Видит | Фотоуроки, вопросы |
| Записываю | Голосовое/текст | Вытаскивает суть, оформляет |
| Делюсь | Публикует | Обратная связь, наставники, площадки |
| Исследую | Задаёт тему | Находит источники, переводит, сохраняет |
