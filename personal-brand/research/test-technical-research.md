# Технический обзор: инструменты для создания теста личности на WordPress

**Проект:** homodivinus.ru  
**Задача:** тест с 15–20 вопросами, несколько вариантов ответов, 8–12 типов результата с описанием и картинкой  
**Дата исследования:** март 2026

---

## Контекст и требования

Тест личности (personality quiz) — это квиз, где итоговый результат определяется не правильными/неправильными ответами, а накопленным «весом» ответов по категориям. Пользователь отвечает на вопросы, каждый ответ вносит баллы в одну или несколько «корзин», и в конце показывается тип с наибольшим количеством баллов.

Ключевые критерии для homodivinus.ru:
- Визуально красиво (результат с картинкой и описанием)
- Без программирования
- Не нужна команда — один человек
- Желательно русскоязычный интерфейс или хотя бы русский контент
- Умеренный бюджет (небольшой проект)

## Приложение: Технические детали реализации

### Как работает тест личности (логика)

```
Вопрос 1: [Вариант A → +2 к типу "Мистик", +1 к типу "Мудрец"]
           [Вариант B → +2 к типу "Воин", +1 к типу "Творец"]
           [Вариант C → +3 к типу "Целитель"]
...
В конце: считаем баллы → показываем тип с максимальным значением
```

Все рекомендованные инструменты поддерживают эту логику.

### Структура результата (пример)

Хороший результат теста личности содержит:
- **Название типа** (н-р: «Архетип Мага»)
- **Картинка** (иллюстрация, символ, портрет)
- **Описание** (3–5 абзацев о свойствах типа)
- **Кнопки шеринга** («Поделиться результатом»)
- **CTA** (н-р: «Узнай больше о своём архетипе» → подписка)

### О русском языке

Все инструменты позволяют писать вопросы, ответы и описания результатов на русском языке. Интерфейс администратора — на английском, но это не критично для одного человека. Сам тест, который видит пользователь — полностью на русском.
