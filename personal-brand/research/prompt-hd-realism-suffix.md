# ✨ HD Realism — стиль-суффикс (для любой новой сцены)

**Статус:** ФИНАЛ, утверждён 17.03.2026
**Назначение:** добавлять к описанию ЛЮБОЙ новой сцены для сохранения стиля HD Realism

## Суффикс для Алисы (RU)

```
тёплая акварельная иллюстрация, cozy style, золотое свечение + искорки, палитра: золото, терракота, кремовый
```

## Суффикс для ChatGPT (EN)

```
warm watercolor illustration on textured paper, soft diffused golden glow like an angel halo, tiny golden sparkles, warm palette of gold terracotta and cream, gentle brushstrokes, quiet contemplative mood, no text
```

## Как использовать

1. Опиши сцену своими словами (например: «мама читает книгу ребёнку перед сном»)
2. Добавь суффикс в конец
3. Получишь сцену в стиле HD Realism

## Фирменный код стиля

- Лёгкие нимбы + золотой свет = homo divinus
- Сакральность почти невидима — если слишком очевидна, исчезает магия
- Формула: полуреалистичная акварельная сцена обычной жизни, в которой мягкий золотой свет и лёгкие нимбы намекают на божественное присутствие
