# 📱 Стратегия соцсетей Homo Divinus

> Создано: 06.03.2026. Обновлять по мере развития.

## Принцип: одна единица контента → адаптация под все платформы

Юля создаёт ОДИН материал → адаптируется под формат каждой площадки.
Не 6 блогов, а один поток в 6 окон.

## Архитектура платформ

```
               homodivinus.ru + Dzen
            (статьи, переводы, глубина, SEO)
                    ↑ анонсы ↓
    ┌───────────────┼───────────────┐
    │               │               │
 Telegram        VK            Pinterest
 (норка,      (эстетика,     (визуальный
 кухня,       реклама,        поисковик,
 свои)        рост)           трафик)
    │               │               │
    └───────┬───────┘               │
            │                       │
         Threads                    │
       (авторское,                  │
        вопросы,              ──────┘
        охват)            VK ❤️ Pinterest-стиль
```

## Роли платформ

| Платформа | Роль | Формат | Аудитория | Рост |
|-----------|------|--------|-----------|------|
| **homodivinus.ru** | Центр, хаб | Переводы статей, авторские статьи, медиа-проект | Поисковый трафик | SEO |
| **Dzen** | Зеркало сайта + SEO | Статьи (копипаст с сайта) | Яндекс-аудитория, РУ | SEO + рекомендации |
| **Telegram** @homodivinus | Личная норка, кухня | Посты из покоя, живое, глубина | Свои, лояльные | Органика, сарафан |
| **VK** | Русская гостиная | Адаптации, Pinterest-эстетика, статьи | Массовая РУ | Реклама, органика |
| **Pinterest** | Витрина, визуальный поисковик | Вертикальные пины, мудборды, Idea Pins | Женщины 25-45, slow living | SEO, органика (пассивно!) |
| **Threads** @chill_yulia | Крыльцо, авторский голос | Вопросы, наблюдения, короткое | Международная | Авторское, communities |

## Связки и воронки

### Перевод статьи (длинный формат)

### Принципы роста
- 5-10 пинов/день (30% свои, 70% курирование)
- SEO в каждом описании
- Вертикальные изображения 1000×1500
- Консистентность > количество
- Результат через 3-6 месяцев
- Проверять тренды: trends.pinterest.com

## Контекст решений

- TG блокируют в России → нужно присутствие на русских платформах (VK, Dzen)
- VK любят Pinterest-эстетику → прямая связка Pinterest↔VK
- В Threads ценят авторское → вопросы, живые наблюдения
- Работа вдолгую. TG может не стать главным — и это ок
- VK = можно запускать рекламу (единственная платформа с доступной рекламой для РУ)
- Instagram = пока на паузе (высокие затраты, Reels). Подключить когда будет контент из Pinterest
