# Блоги

Архив постов из Telegram-каналов членов семьи.

## Назначение

Эта папка — архивный поток публичной рефлексии:
- Посты хранятся в хронологическом порядке
- Имеют самостоятельную ценность как дневник мыслей
- Можно перечитывать, искать инсайты, отслеживать эволюцию

## Формат постов

Все посты сохраняются в едином формате:

```markdown
# Заголовок поста

Источник: Telegram @канал
Дата поста: 2025-10-14
Автор: Имя

Текст поста...
```
