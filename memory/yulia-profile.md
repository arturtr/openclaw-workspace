## Контент-стратегия (22.02.2026)
- Поток постов = норма (III дом, 3 планеты), но нужен РИТМ: глубокие + лёгкие + паузы
- Пауза = тоже контент. Без пауз нет мелодии
- Голос/видео > текст. Луна I = через голос чувствуют мгновенно
- Перед постом: 30 сек тишины → «что я чувствую?» → писать из тела, не из головы
- Если не чувствуешь — не пиши. Тишина лучше пустого поста
- Формат подруги за чаем — Юля сама подтвердила, чай буквально часть жизни и бизнеса
- Чай как мост к встрече: «хочешь поговорить за чаем?» снимает барьер
- Соцсети = комнаты одного дома: Telegram=кухня, ВКонтакте=гостиная, Дзен=веранда, Instagram=окно, YouTube=приглашение за стол

- Кухня = Telegram (80% энергии). Окно = Instagram или ВК (20%). Остальное = автопосты или пауза
- Привлечение: курсы (представляется → подписчики, уже работает!) + сарафан (#DeepMeet) + гостевые эфиры + коллаборации
- Аудитория приходит к тем, кто ДОМА, а не к тем, кто бегает по площадкам
