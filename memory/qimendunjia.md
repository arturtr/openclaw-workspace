# 奇门遁甲 (Qí Mén Dùn Jiǎ) — Таинственные Врата

> Справочник для Кузи. Использовать когда: Артур спрашивает про время действия, направление движения, стратегические решения. Сергей Андреев — чайный учитель Артура — преподаёт эту систему. Связывать с чаем: время сбора, сезон горы, выбор дня для чайной встречи.

---

## Три тайные системы 三式

Ци Мэн — часть триады 三式 (sān shì). Три уровня одной реальности:

| Система | Уровень | Вопрос |
|---------|---------|--------|
| 太乙 Tài Yǐ | Макро — эпохи, государства | «В какой эре мы живём?» |
| 六壬 Liù Rén | Мезо — события, ситуации | «Что произойдёт в этой ситуации?» |
| 奇门遁甲 Qí Mén | Тактика — часы, дни, направления | «Когда и куда действовать?» |

Если 易经 отвечает «какова природа момента» — 奇门遁甲 отвечает «как действовать в этом моменте практически».

---

## Структура системы

Название расшифровывается послойно:

- **奇 Qí** — «Три чудесных» ствола: 乙 (yǐ) · 丙 (bǐng) · 丁 (dīng)
- **门 Mén** — «Восемь Врат»: маршруты энергии и действия
- **遁甲 Dùn Jiǎ** — «Сокрытый Цзя»: ствол 甲 (старший брат Дерева, начало) всегда скрывается за другим

Карта — **盘 pán** — строится для конкретного момента времени. Девять дворцов (九宫) из 洛书 (Luòshū), в каждом одновременно четыре слоя:

| Слой | Элемент | Что описывает |
|------|---------|--------------|
| 星 Xīng | Девять Звёзд | Природа места/энергии |
| 门 Mén | Восемь Врат | Тип действия / маршрут |
| 神 Shén | Восемь Деятелей | Силы судьбы, союзники/враги |
| 干 Gān | Небесные Стволы | Временны́е качества |

## 八门 Bā Mén — Восемь Врат

Центральный элемент для принятия решений. Каждые Врата — тип энергии действия:

| 门 | Пиньинь | Значение | Качество |
|----|---------|----------|---------|
| 休门 | Xiū | Покой | Отдых, восстановление, вода |
| 生门 | Shēng | Жизнь | **Самые благоприятные.** Новые начала, здоровье, бизнес |
| 伤门 | Shāng | Рана | Активное действие, спор, охота |
| 杜门 | Dù | Заслон | Укрытие, уход, защита |
| 景门 | Jǐng | Вид | Слава, письмо, путешествие, огонь |
| 死门 | Sǐ | Смерть | Завершение, захоронение, опасность |
| 惊门 | Jīng | Потрясение | Неожиданность, суд, страх |
| 开门 | Kāi | Открытие | Инициатива, власть, новый путь |

**Благоприятные Врата:** 生 · 休 · 开 (в этом порядке). Для большинства дел — искать 生门.

## Применение

### Выбор времени (择时 zé shí)

Строят карту на нужный момент → смотрят, какие Врата активны в нужном дворце → если 生门 или 开门 — действуй. Если 死门 или 惊门 — отложи.

### Выбор направления (择向 zé xiàng)

Буквально: в каком направлении двигаться, чтобы поймать попутную энергию. Применимо к: поездке, переезду, деловой встрече, посеву, сбору чая.

### Чай и 奇门

Связь не случайная:
- **Время сбора**: 明前茶 (до Цинмин) — не просто традиция. Это период, когда карта 春分→清明 даёт максимум Дерева + ранней Воды → самый мягкий лист.
- **Направление горы**: южный склон (朱雀 = Огонь = активация) даёт более ароматный чай. Северный (玄武 = Вода = укрытие) — более глубокий, медленный.
- **День встречи**: выбор дня по Ци Мэн — практика профессиональных чайных мастеров.

## Связь с Ба Цзы Артура

Дневной ствол Артура — **丙 (bǐng)**, Огонь Солнца. Это один из **трёх Ци** (奇) в системе: 乙 · 丙 · 丁. В 奇门遁甲 ствол 丙 означает видимость, ясность, харизму Солнца — тот, кто освещает.

Когда в карте момента активен 丙 — Артуру это попутный ветер.

## Для Кузи: как использовать

- При вопросе «когда лучше начать проект / провести встречу» — можно упомянуть Ци Мэн как инструмент выбора времени
- При разговоре о чае с Сергеем Андреевым — система общая, язык понятен
- При обсуждении 节气 — Ци Мэн тоже работает с сезонными картами (時盤 shí pán)
- Не строить карты самостоятельно без запроса — это требует специализированных инструментов
