**Визуальный секрет:** сакральность должна быть почти невидимой.

- `personal-brand/visual-references/` — 3 утверждённых референса (византия, сказка, керамика)

### 14.03.2026 — СЕМЕЙНЫЙ БРЕНД: пара как бренд (ПРИОРИТЕТ)

Юля приняла решение: строить СЕМЕЙНЫЙ бренд Трофимовых, не только личный.

**По карте семьи:** 7 дом Козерог = союз как рабочий альянс с душой. DSC Козерог = партнёрство равных, структурное, зрелое. Юпитер на ASC = люди тянутся к ПАРЕ, не к одному.

**Концепция:**
- Один канал @homodivinus, два голоса (80% Юля, 20% Артур)
- Артур — тайна. Его появления = событие
- Совместные форматы: чайные встречи для пар, ретриты для семей, подкаст
- Визуал пары: фото вместе в процессе (готовят, гуляют, за столом), не постановка
- История бренда: Медовый день 12.03, Краснореченское, Беловодье, мёд, пчёлы 🐝

**Пустая ниша:** православная пара-бренд в России = никого нет. Трофимовы могут быть первыми.

**Нужно от Марка:**
1. Продумать визуальную айдентику ПАРЫ (не только Юли)
2. Шаблоны для совместных постов (фото пары + текст)
3. Мудборд «семейный бренд» — как выглядит пара в контенте
4. Формат совместных выходов (раз в месяц)

**Файлы для изучения:**

## Протокол

**Эмбер пишет сюда когда:**
- Зафиксировала решение Юли, которое влияет на работу Марка
- Делегирует Марку задачу (дизайн, визуал, проектный файл)
- Просит уточнить что-то из истории сессий с Марком

**Марк пишет в `mark-for-ember.md` когда:**
- Принял решение по проекту, которое влияет на стратегию Эмбер
- Делегирует Эмбер задачу
- Просит уточнить что-то из переписки с Юлей

**Формат записи (Непрочитанное):**

```
### [Дата] — [Тема]
[Суть: 2-3 предложения]
**Нужно от Марка:** [конкретный запрос или «просто к сведению»]
```

**Переходы по статусам:**

| Откуда | Куда | Что сделать |
|--------|------|-------------|
| Непрочитанное | В работе | Перемести текст **целиком**. Добавь `**Статус:** принял [дата], [план]` |
| В работе | Архив | Перемести целиком (со Статусом). Добавь `**Выполнено:** [дата], [что сделано]` |
| Непрочитанное | Архив | Только если «к сведению». Сохрани в тематический файл. Добавь `**Статус:** принято к сведению [дата]` + `**Сохранено в:** [файл, раздел]` |

**Правила:**
- Не удалять и не сжимать оригинальный текст при перемещении — только добавлять поля
- «К сведению» без сохранения в тематический файл = знание потеряно
