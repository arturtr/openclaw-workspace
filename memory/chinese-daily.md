# 🈶 Иероглиф дня — правила, прогрессия, журнал

> Единый источник правды для процесса обучения. Читается кроном каждое утро.

## Правила

1. **Один новый в день.** Не больше (кроме пакетов простых).
2. **Строго по порядку уровней.** Не перескакивать.
3. **Строгое правило компонентов:** составной — ТОЛЬКО если ВСЕ его видимые компоненты в журнале. Разложить → проверить каждый → если нет — сначала компонент.
4. **Словосочетания** (忍耐, 坚忍) — только если оба иероглифа пройдены по отдельности.
5. **Пакетная отправка:** примитивы ≤3 черт можно по 2-3 в одном сообщении, каждый с полной карточкой.
6. **Не повторять** из последних 60 дней (кроме SRS-повторений).
7. **Тег #иероглиф — ПЕРВАЯ СТРОКА** сообщения. Отдельное сообщение через message/send.
8. **НЕ давать мини-задания** с непройденными иероглифами. Проверять по журналу.
9. **НЕ писать «ты уже знаешь X»** если X не пройден формально (знаком ≠ пройден).
10. **SRS-повторение:** включить 2-3 старых из chinese-srs.md (где Следующий ≤ сегодня) в начало сообщения.
11. **Контекстные предложения:** ТОЛЬКО из пройденных иероглифов (журнал) + текущий новый. Непройденные слова — пиньинь, не иероглифы.
12. **Тройной перевод:** каждый иероглиф — 汉字 → русский → English. Всегда.

## Приоритет выбора

1. SRS-повторения (если есть due в chinese-srs.md)
2. Базовые ключи (Уровень 1) — пока не пройдены все 43
3. Составные (Уровень 2) — когда компоненты готовы
4. Тематически по контексту (чайный день → Чай, Ба Цзы → стихии)
5. По порядку тем: Воля → Чай → ТКМ → Ба Цзы → Техника

## Формат карточки (утреннее сообщение)

```
#иероглиф

汉字 [большим шрифтом]
pīnyīn 🔴/🟠/🟢/🔵 (тон) · русский · English

🧩 Состав: [из каких частей, если составной]
💡 Мнемоника: [история/образ для запоминания]
📜 Этимология: [oracle bone → seal → modern, кратко: как рисунок стал символом]

✍️ Порядок черт:
1. ...
2. ...

🔊 Фонетическая серия (если есть):
[звуковой компонент] даёт звук X в: [список иероглифов]

📝 Предложение:
[из пройденных иероглифов + текущий. Непройденные — пиньинь]
(pīnyīn всего предложения)
русский перевод · English translation

🔍 Где встретишь: [контекст из жизни Артура — чай, ТКМ, Ба Цзы]
↔️ Антоним: [если есть]
📋 Мини-задание: [практика с пройденными иероглифами]
```

**Тоновые эмодзи:**
- 🔴 1й тон (ровный): shān, jīn
- 🟠 2й тон (восходящий): rén, mén
- 🟢 3й тон (нисходяще-восходящий): shuǐ, huǒ
- 🔵 4й тон (нисходящий): mù, dào
- ⚪ нейтральный: de

**По понедельникам** — добавить блок «📋 Задание на неделю»:
- Привязано к пройденным за прошлую неделю
- Примеры: «Найди 茶 на упаковке», «Запиши все знакомые иероглифы на пакете X»
