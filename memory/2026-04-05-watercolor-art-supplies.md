### 2. АКРИЛ (для холста и тренировок)

У тебя уже есть акрил — **докупи только:**
- **Белила титановые** (если нет) — для велатуры (молочных вуалей)
- **Золотой акрил** или **поталь** — для нимбов и акцентов

### 3. БУМАГА (критично!)

Для сфумато бумага важнее красок. **Хлопок 300 г/м², среднее зерно (cold press)**. Хлопок впитывает воду медленнее = больше времени на работу мокрым по мокрому.

- [Ozon: Гамма Студия, 100% хлопок, 300 г/м², А4, 10 листов](https://www.ozon.ru/category/hlopkovaya-bumaga-dlya-akvareli/) — ~400-600₽. Лучший вариант для начала

**Для миниатюр на рваной бумаге** (как в твоих референсах): можно просто рвать эти листы на кусочки 5×7 см. Рваный край = часть эстетики.

### 4. КИСТИ

|-------|-------|-----------|
| **Плоская широкая №16-20** | Заливки фона, «вуали» | Синтетика или белка. ~200-400₽ |
| **Круглая №8-10** | Силуэты, средние формы | Колонок или белка. ~300-500₽ |
| **Круглая №2-4** | Детали, контуры, леттеринг | Колонок. ~200-300₽ |
| **Флейц (широкая плоская) №40-50** | Смачивание бумаги целиком | Любая мягкая. ~150₽ |

- [Ozon: кисти для акварели набор](https://www.ozon.ru/search/?text=кисти+акварель+набор+колонок&from_global=true) — набор 5-7 кистей ~500-1000₽
