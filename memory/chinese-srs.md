# SRS — интервальное повторение иероглифов

## Принцип

Каждый пройденный иероглиф проходит 5 интервалов повторения:

| Уровень | Интервал | Значение |
|---------|----------|----------|
| 1 | 1 день | первое закрепление |
| 2 | 3 дня | краткосрочная память |
| 3 | 7 дней | среднесрочная память |
| 4 | 14 дней | долгосрочная память |
| 5 | 30 дней | финальная проверка |
| ✅ | — | выучен, перемещён в «Выучены» |

Утреннее сообщение включает 2-3 иероглифа из колонки «Следующий ≤ сегодня».
Формат повторения: 汉字 → пауза → вспомнил? → pīnyīn + значение + English.
После каждого урока крон повышает SRS на 1 и пересчитывает дату.
