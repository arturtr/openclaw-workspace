# 🇨🇳 Словарик иероглифов Артура

> Учим потихоньку, по контексту. Не зубрёжка — а встретил → записал → запомнил.

## Методика (предложена Кузей 27.02.2026)

**Главный принцип:** не «учу китайский» — чайный мастер читает свои упаковки. Это часть ремесла.

**Формат:** всегда тройка — 汉字 → русский → English (с 13.03.2026, расширяем и английский словарный запас)

### Письменная практика — 5 минут в день

**1 иероглиф в день.** Не больше. 20 в день = бросишь через неделю. Один = будешь делать год.

| Когда | Что | Время |
|-------|-----|-------|
| Утро (сводка) | Прочитал иероглиф дня | 1 мин |
| Перерыв/после обеда | Прописал в прописях (8–10 раз) | 3–5 мин |
| Вечер (по желанию) | Вспомнил без подсказки | 1 мин |

**Как писать:**
- 3 раза по образцу, последние 3–4 — по памяти
- Рядом подписать пиньинь и значение от руки
- Соблюдать порядок черт: ↓ сверху вниз, → слева направо, сначала рамка → содержимое → закрыть дно

### Запоминание — связи, не зубрёжка

- **Радикалы, не иероглифы.** 20 ключей → 80% иероглифов становятся логичными
  - 水/氵(вода) · 火/灬(огонь) · 木(дерево) · 金/钅(металл) · 土(земля) — уже знакомы по У Синь
  - 口(рот) · 手/扌(рука) · 心/忄(сердце) · 人/亻(человек) · 目(глаз)
- **Истории, не повторения.** 热 = рука над огнём → горячо. Один раз представил — запомнил.
- **Правило 1-1-1:** встретил → записал → прописал → произнёс вслух. Три касания = запомнил.

### Бумажный словарик (опционально)

Маленький блокнот по темам (не по алфавиту): 🍵 Чай → 🏥 ТКМ → 📱 Техника → 🏠 Быт.
Формат страницы: иероглиф · пиньинь · перевод · прописать 3 раза · пара примеров · мнемоника.
Носить с собой — встретил на упаковке → записал.
