# Библиотека курсов

Конспекты онлайн-курсов Юли. Каждый курс — отдельная папка с полной структурой.

## Структура курса

```
courses/
  <slug-курса>/
    README.md              — описание курса, статус прохождения, оглавление
    <NN-slug-блока>/
      README.md            — описание блока
      <NN-slug-урока>.md   — конспект урока
      assets/              — презентации, изображения, доп. материалы
```

## Формат конспекта урока

Каждый файл урока содержит:

1. **Шапка** — название, блок, дата прохождения, ссылки на видео
2. **Ключевые понятия** — термины и определения из урока
3. **Конспект** — структурированное содержание урока
4. **Материалы** — ссылки на презентации, доп. файлы
5. **Вопросы для рефлексии** — что обдумать после урока
6. **Связь с практикой** — как это применимо к работе Юли
