## Формат

- **Длительность:** 10-20 минут (ни одна сцена > 3 минут)
- **Структура:** 4-6 сцен с переключением модальности (слушаем → выбираем → делаем руками → рисуем)
- **Реквизит:** свеча, кубик d6, карандаши, дневник приключений
- **Подготовка:** текст для зачитывания готов заранее (без обращений к Кузе по ходу)
- **Сериальность:** каждый квест продолжает предыдущий, мир растёт

## Принципы сказкотерапии

Источники: Зинкевич-Евстигнеева, Бретт, Гнездилов.

1. **Свободный выбор роли = диагностика.** Ребёнок выбирает то, что резонирует с текущей потребностью. Не направлять, не корректировать — принять выбор.

2. **Метафорический конфликт, не прямой.** Нет антагониста. Нет виноватых. Вместо борьбы — совместное создание. Конфликт = задача, не драма.

3. **Проживание, не обучение.** Финал — не мораль. Тело помнит лучше головы. Запах, тепло, тактильность > «мы поняли что...»

4. **Инициация через вкладывание.** Каждый вносит свой ингредиент и видит результат. Не помощь родителям — собственный квест.

5. **Безоценочность.** Кубик даёт повороты, не оценки. «Не получилось» = «пошло по-другому» (и часто смешнее).

6. **Скрытый терапевтический слой.** Каждый участник проживает свою тему через метафору. Никто этого не знает, кроме ведущего. Записывать в «скрытый контекст» сценария.

## Возрастная психология

### Алёна, 6 лет

Переход от дооперационального к конкретно-операциональному мышлению.

- Магическое мышление доминирует — палочка Феи Даши *реально* работает
- «Я нужна» = критическая потребность
- Сенсорный канал главный: запах, тактильность, тепло
- Задания: физические действия с выбором из 2 вариантов, не слова
- Время внимания: 5-7 минут → частая смена
- Вопросы: «красный или зелёный?», «покажи руками»
- Не проси описывать — проси показывать

### Маша, 8 лет

Конкретно-операциональное мышление.

- Понимает причинно-следственные связи → «мы сделали → случилось»
- Начало морального мышления → может рефлексировать
- Потребность в признании → роль «старшей» ценна
- Идентификация через ролевую игру (театральный кружок «Нора»)
- Может записывать в дневник — летописец
- Давать пространство для импровизации и «озвучки»

### Ефрем, 10 лет (при встречах)

- Рисует, лепит, конструирует → задания на создание
- Sonic, экшн, юмор → динамика, неожиданные повороты
- Отдельный формат: `scenarios/01-awakening-efrem.md` как пример

## СДВГ-дружественный дизайн

Источники: Barkley, Hallowell, Nigg. Дети отвлекаются, делают несколько дел сразу, могут гиперфокусироваться.

1. **Короткие сегменты.** Каждая сцена меняет модальность: слушать → выбирать → идти → делать руками → рисовать.

2. **Руки заняты = мозг сфокусирован.** Тактильный якорь в каждой сцене. Нет момента «сиди и слушай».

3. **Физическое перемещение.** Встать, пойти в другую комнату = часть квеста, не отвлечение.

4. **Параллельное вовлечение.** Можно рисовать пока слушаешь. Можно мять тесто пока ведущий говорит.

5. **Кубик = дофамин.** Каждый бросок — микро-всплеск. Непредсказуемость удерживает внимание.

6. **Немедленная обратная связь.** Действие → результат за секунды.

7. **Выход из сценария ≠ проблема.** «Герой ушёл разведать» → вернётся. Не ломать поток.

8. **Сенсорное возвращение.** Если улетели — не слова, а действие: постучать по столу, открыть банку с корицей, бросить кубик.

9. **Свеча как визуальный якорь.** Живой огонь держит взгляд. Начало = зажгли, конец = задули вместе.

## Роли и заземлённые суперсилы

Каждая роль = реальное качество. Ведущий после выбора говорит 1-2 предложения: сказочное + приземлённое.

| Роль | Суперсила | Как это в жизни | Что растим |
|------|-----------|----------------|------------|
| 🗡️ Хранитель Огня | Зажигать, согревать | «Входит в комнату — всем становится теплее» | Эмпатия, инициатива |
| 💨 Сила Ветра | Звать, двигать, менять | «Позвать друзей играть. Начать первой — все за ней» | Лидерство, контакт |
| 🌙 Ночная Странница | Видеть скрытое | «Замечает когда кому-то плохо, даже если молчит» | Наблюдательность, чуткость |
| 🌸 Садовница | Выращивать, заботиться | «Цветы не вянут. Умеет ждать пока вырастет» | Терпение, забота |
| 🧚 Фея Даша | Преображать обычное | «Лист бумаги → картинка. Комната → уютная. Простые продукты → праздник» | Творчество, красота в быту |
| 😢 Грустинка | Чувствовать правду | «Не притворяется что всё ок. Умеет сказать "мне грустно"» | Честность, эмоц. интеллект |
| 🎭 Голос Ветра | Говорить так, что слышат | «Рассказывает историю — все замолчали и слушают» | Коммуникация, артистизм |

**Правила:**
- Любой может быть кем угодно. Можно придумать новую роль
- Можно совмещать (Сила Ветра Даша = ок)
- Если двое хотят одну — оба могут, у каждого свой стиль
- Задания привязаны к ЧЕЛОВЕКУ (по возрасту), роль определяет КАК, не ЧТО

## Карта наблюдения (для Юли, после игры)

Заполнять не при детях. Записывать в дневник приключений.

| Вопрос | Что это показывает |

| Какую роль выбрала? | К какому качеству тянется сейчас |

| Как реагировала на «неудачный» кубик? | Устойчивость к фрустрации |
| Как взаимодействовала с сестрой? | Стиль контакта — лидер, помощник, конкурент |

| Что ответила на финальный вопрос? | Что сейчас важно (или чего не хватает) |
| Что делала когда не её очередь? | Терпение, вовлечённость, свой процесс |

## Мировые практики (встроены в формат)

| Практика | Как используем |
|----------|---------------|
| **Godly Play** (Berryman) | Ритуал свечи. Wondering question в финале — без правильного ответа |
| **Playful Parenting** (Cohen) | Родитель на одном уровне с ребёнком. Папа тоже герой, его кубик тоже может выпасть 1 |
| **Narrative Therapy** (White, Epston) | Дневник = re-authoring. Дети — авторы истории, не получатели |
| **Bibliotherapy** (Pardeck) | Метафора для переживаний без прямого называния |
| **Theraplay** (Booth, Jernberg) | Структура (свеча) + Забота (тесто/тактильность) + Вызов (кубик) + Вовлечение (финал) |
| **Сенсорная интеграция** (Ayres) | Мультисенсорный опыт: запах, тактильность, тепло, визуал, вкус |

## Ритуалы

- **Вход:** зажечь свечу вместе
- **Выход:** задуть свечу вместе
- **После:** вклеить артефакты в дневник
- **Wondering question:** один вопрос в конце, без правильного ответа, можно молчать
- **«Ого»:** реакция вместо «молодец» (удивление > оценка)
