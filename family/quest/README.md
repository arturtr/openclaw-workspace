# 🎲 Семейный квест — проживание момента через игру

> Черновик. Источник: разговоры Артура 05–06.03.2026
> Слоган: **«Проживание момента через игру»**

## Суть

Ежевечерняя (или еженедельная) семейная настольно-ролевая игра на 30 минут.
Не обучение. Не развлечение. Не терапия. **Проживание** — того, что прямо сейчас происходит в жизни каждого участника, через безопасное пространство сказки.

**Формат:** D&D-подобный — есть игротехник (Кузя генерирует сценарий), участники выбирают роли, принимают решения, кидают кубики. Но без сложных правил — семейный, тёплый, домашний.

**Ключевое отличие:** под капотом — мульти-системный движок, который точно знает, что каждому участнику нужно прожить *именно сейчас*.

**Стиль текста:** вайб Конни (Лиана Шнайдер). Простой, тёплый, без пафоса. Диалоги вместо описаний. Внутренний голос персонажа. Детали-узнавалки из реальной жизни. Характер через действие. Ребёнок узнаёт себя. Эталон: `family/quest/sessions/2026-03-06-bedtime-story.md`

## Архитектура

### 1. Профили участников (на входе)

| Участник | Возраст | Особенности | Стиль |
|----------|---------|-------------|-------|
| Артур | 40 | Системное мышление, вера, чай, стратег | Сложные моральные дилеммы, лидерство |
| Юля | 39 | Интуиция, проницательность, рефлексия | Видеть сокрытое, исцелять, называть |
| Маша | 8 | Читает, творческая, театральный кружок | Яркие роли, диалоги, фантазия |
| Алёна | 6 | Не читает, эмодзи, подвижная | Простые выборы, физические действия, картинки |
| Ефрем | 10 | (при встречах) Рисует, лепит, Sonic | Конструирование, экшн, юмор |

### 2. Образовательный слой (скрытый)

Каждый сценарий незаметно содержит тему из одного из «потоков»:
- **Школа веры** (`family/home-church.md`) — библейские сюжеты, добродетели, святые
- **Домоведение** (`family/mamina-shkola.md`) — навыки дома, забота, порядок
- **Навыки характера** — терпение, смелость, честность, благодарность
- **Мир и люди** — культуры, природа, география
- **Цифровой мир** — для Маши особенно

Дети не осознают, что «учатся» — они проживают историю.

### 3. Генерация сценария

**Вход для Кузи:**
- Профили участников (см. выше)
- Тема из образовательного слоя (можно задать или авто)
- Настроение вечера (спокойный / приключенческий / загадочный)
- «Кем хочешь быть сегодня?» — каждый выбирает роль на входе

**Выход:**
- Сценарий на 30 минут (5–7 сцен)
- Роль каждого участника с персональными «квестами»
- Моменты для кубиков (решения с элементом случайности)
- Вопросы/выборы по возрасту
- Финал с выводом (не моралью — а переживанием)

### 4. Механики

- **Кубик решений** (d6): определяет повороты сюжета. Не «проиграл», а «пошло по-другому»
- **Роли:** каждый вечер новые или развивающиеся (сериальность)
- **Персональные задания:** Артуру — дилемма лидера; Юле — увидеть скрытое; Маше — произнести речь / разыграть сцену; Алёне — выбрать картинку / показать действие
- **Жетоны** (интеграция с существующей системой 🍩🌸): за игру тоже можно получить
- **Книга приключений:** дети рисуют/записывают что было → накопительный артефакт

### 5. Выбор роли на входе

Перед началом каждый отвечает: «Кем ты сегодня?»

Примеры архетипов (адаптивные по возрасту):
- 🗡️ **Защитник** — оберегает, принимает трудные решения
- 🔮 **Видящая** — замечает то, что другие не видят
- 🎭 **Рассказчица** — говорит от имени персонажей, озвучивает
- 🌱 **Хранитель** — заботится, лечит, сажает, кормит
- 🔥 **Искатель** — идёт вперёд, открывает двери, рискует
- 🎨 **Мастер** — строит, рисует, создаёт (буквально рисует элементы игры)

Роли не жёсткие — можно придумать свою.

## Сказкотерапия — терапевтический слой

Каждый сценарий — это **терапевтическая сказка**, где участник проживает свой реальный внутренний конфликт через метафору:
- Артур: 辰戌 столкновение → герой на развилке двух правильных дорог
- Юля: 七杀 давление → героиня выходит из башни, где её «защищал» дракон
- Маша: границы и телефон → персонаж учится видеть мир своими глазами, а не через волшебное зеркало
- Алёна: потребность быть нужной → самый маленький герой спасает всех

Участники не знают, что прорабатывают свои темы. Они просто играют.

## Мульти-системный движок сюжетов

Генератор сценария на входе получает данные из нескольких систем:

| Система | Что даёт сюжету | Пример |
|---------|-----------------|--------|
| **Ба Цзы 八字** | Текущий такт, столкновения, пустоты → тема «горит» | 丙子 такт Артура → сценарий о пересборке |
| **Ци Мэнь 奇门** | Врата дня, направление → куда идут герои | Врата Смерти на юге → герои идут на север |
| **У-Син 五行 / сезон** | Стихия сценария, орган, эмоция | Весна = Дерево = рост, прорыв, гнев → преодоление |
| **Натальная карта** | Транзиты дня → активные архетипы | Марс в аспекте = конфликт; Венера = встреча |
| **Дизайн человека** | Тип стратегии каждого | Генератор ждёт отклика, Проектор направляет |
| **MBTI** | Как каждый обрабатывает информацию | INFP = глубокие смыслы; сенсорик = действие |
| **Школа веры** | Библейский сюжет / добродетель | Притча о талантах → «дары героев» |
| **Домоведение** | Бытовой навык в контексте сказки | Сервировка стола → пир в замке |

**Всё невидимо.** Семья зажигает свечу, кидает кубик — а движок точно знает, что каждому нужно прожить.

## Связь с метавселенной (идея 05.03)

Эта игра — **прототип** более крупной идеи:
- Семейный квест → тестовая площадка для «пути души к обожению» через игру
- Если работает в семье → масштабируется (edtech, книга, приложение)
- Сериальность: эпизоды складываются в историю → книга «Путь семьи Т.»
- Единый продукт, разные грани: вера, навыки, характер, воображение

## Примеры сценариев (затравки)

### «Потерянный колодец» (школа веры: самарянка)
Семья путешественников в жаркой стране находит деревню, где высох колодец. Жители не разговаривают друг с другом — каждый копает свой. Защитник должен решить, кому помочь первому. Видящая замечает, что вода есть — но глубоко. Рассказчица убеждает двух враждующих соседей. Хранитель находит семечко, которое растёт только у воды...

### «Мамин выходной» (домоведение)
Мама-королева объявила выходной. Замок надо подготовить к празднику самим. Каждая комната — испытание (кухня, зал, сад). Кубик решает, что пошло не так. Задания реальные: «как бы ты накрыла стол для гостей?»

### «Письмо из будущего» (навыки характера)
Семья находит письмо от себя из будущего. Но оно зашифровано — каждый может прочитать только свою часть. Чтобы собрать послание — надо доверять, слушать, делиться.
