# 🏫 Мамина школа — ежедневные уроки для девочек

> «Фильтр для плохого. Линза для хорошего.» — Юля, 05.03.2026

## Кому
- **Маша** (8 лет, 2 класс) — читает, осваивает интернет, театральный кружок
- **Алёна** (6 лет, детсад) — не читает, ориентируется по эмодзи, подготовка к школе

## Когда
- **Ежедневно в 15:00** (Хабаровск, GMT+10) — Эмбер присылает урок
- Будни: ~1 час вечером свободного времени
- Выходные: больше времени, можно задания подлиннее
- Формат: короткий, без назидания, через вдохновение

## Рубрики (чередовать)

### 📱 Цифровой мир
- Интернет-этикет
- Цифровая безопасность
- Рекомендации игр (Android, русский язык, в т.ч. библейские)
- Правила экранного времени

### 🏠 Домоведение для девочек
- Посильные навыки по дому (по возрасту)
- Готовка вместе с мамой (простое)
- Забота о пространстве, красота в мелочах
- Сервировка, порядок, «мамина помощница = привилегия»

### ✝️ Девочки-христианки
- Истории святых (коротко, ярко, с картинкой/видео)
- Библейские героини
- Добрые дела как привычка
- Молитва своими словами

### 🎨 Хобби и творчество
- Дзендудлы, зен-арт, раскраски
- Бисероплетение, фенечки
- Аппликации, коллажи (современные!)
- Рисование, лепка
- Каждый раз — новое хобби попробовать

### 🌍 Мир и люди
- Страны, культуры, традиции
- Природа, животные
- Профессии, вдохновляющие люди
- «А ты знала, что...?» — удивительные факты

### 🎬 Рекомендации контента (ТОЛЬКО с проверенными ссылками!)
- YouTube / RuTube / VK Видео — по возрасту
- Кинопоиск — фильмы и мультфильмы
- Для Маши (8): уход за собой, ценности, хобби, мир
- Для Алёны (6): развивающее, доброе, красивое

### 💪 Навыки характера (через вдохновение, не назидание!)
- Концентрация, усидчивость
- Аккуратность
- Внимание к другим, помощь
- Благородство
- Трудолюбие
- Любопытство, интерес к жизни

## Задания для Юли
- Как коуч для детей: наблюдения, вопросы, эксперименты
- Как женщина: свои хобби рядом с детьми (рисование, раскраски, дзендудлы)
- НЕ: вышивать, клеить, лепить поделки (не её)
- ДА: коллажи, зен-арт, готовка, бисер, рисование

## Задания для Артура
- Отцовские моменты: что может только папа
- Совместные активности

## Школа родителей (Юля + Артур)
- Микро-уроки: 1 принцип / 1 практика / 1 наблюдение
- Источники: Войджитски, Крыгина, «О семье и воспитании» (Ведяничев)
- Воспитание из сильных сторон родителей

## Принципы
- **Без назидания и клише** — через вдохновение
- **Почувствовать и найти в себе** — не навязывать
- **Готовить к старшему возрасту** — трудолюбие, цель, смысл, любопытство
- **Крутые, с которых берут пример** — не «послушные», а вдохновляющие
- **Окружить добротой и любовью** — атмосфера > контроль
- **Все ссылки проверять!** Только рабочие, кликабельные
- **Android** — для рекомендаций игр/приложений
