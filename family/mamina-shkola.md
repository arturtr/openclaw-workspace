# 🏕 Детская школа Homo Divinus

> «Фильтр для плохого. Линза для хорошего.» — Юля, 05.03.2026
> Продукт в стиле Why Not Month: ежедневная образовательная программа для детей

## Большая идея

Взращивание здорового, проактивного, психологически устойчивого, трудолюбивого, благородного поколения. Образование и воспитание нового поколения людей с учётом самобытности семей, уникальности детей и родителей.

Как Артек, но с более глубинными, человечными, нравственными основами.

## Кому
- **Маша** (8 лет, 2 класс) — читает, осваивает интернет, театральный кружок
  - Размышление — **зона роста**, не сильная сторона. Развивать мягко через задания
  - Часто подходят те же задания, что и для Алёны
  - Добавлять ⭐ задание со звёздочкой: посложнее, с доп. шагами, исследованием, поиском информации
  - Квесты на несколько дней — подходят!
- **Алёна** (6 лет, детсад) — не читает, ориентируется по эмодзи, подготовка к школе
  - Схватывает на лету
  - Задания через игру, рисунок, тело

## Когда
- **Ежедневно** — Эмбер присылает материал на день, с регуляцией нагрузки
- Ежедневное может жить ТОЛЬКО если занимает совсем немного времени

### Нагрузка по дням

| День | Маша | Нагрузка | Формат |
|------|------|----------|--------|
| **Пн** | Уроки | 🟡 Средняя/низкая | Короткое задание после уроков |
| **Вт** | Приходит поздно + уроки | 🟢 Расслабленный | Минимальное, лёгкое |
| **Ср** | Уроки | 🟡 Средняя/низкая | Короткое задание после уроков |
| **Чт** | Прогулка | 🟢 Пассивное | Что-то с учётом прогулки/гаджетов, сказка перед сном |
| **Пт** | Приходит поздно + прогулка | 🟢 Пассивное | Что-то с учётом прогулки/гаджетов, сказка перед сном |
| **Сб** | Свободный | 🟡 Средняя (первая половина дня, ненадолго) | Короткое дело или подготовка к выходу |
| **Вс** | Свободный | 🔴 Выход + семейное | ВЫХОД из дома всей семьёй + школа родителей вечером |

- **Потестим — отрегулируем.** Нагрузка живая, не высечена в камне
- Формат: короткий, без назидания, через вдохновение

- Официальный запуск — после утверждения программы

## Методология

### Спираль месяца
Каждый месяц = одна генеральная тема. 4 недели углубляют её:
1. **Замечаю** (наблюдение, внимание)
2. **Чувствую** (внутренний мир, эмоции)
3. **Пробую** (действие, руки, опыт)
4. **Дарю** (щедрость, служение, плод)

Путь homo divinus в миниатюре: пробудиться → почувствовать → проявиться → послужить.

### Привязка к календарю
- Церковный календарь (посты, праздники, святые)
- Природный календарь (сезоны, 节气)
- Семейный календарь (события, дни рождения)

### Принцип заданий
- Базовое задание = одинаковое для обеих
- ⭐ Звёздочка = усложнение для Маши (или кто захочет): больше шагов, исследование, поиск, квест
- 🌈 Квесты на несколько дней — сквозные задания через неделю

## Рубрики (ритм по дням недели)

### Пн — 📱 Цифровой мир (активное)
- Интернет-этикет, цифровая безопасность
- Рекомендации игр (Android, русский язык)
- Фото-задания, цифровые навыки

### Вт — 🏠 Домоведение (активное)
- Посильные навыки по дому (по возрасту)
- Готовка вместе с мамой (простое)
- Сервировка, порядок, красота в мелочах

### Ср — 🎨 Хобби и творчество (активное)
- Зен-арт, дзендудлы, коллажи
- Бисероплетение, рисование
- Мама тоже делает своё рядом!

### Чт — 🌍 Мир и люди (ПАССИВНОЕ — девочки гуляют!)
- Короткое видео / подкаст / аудиоистория перед сном
- «А ты знала, что...?» — удивительные факты
- Страны, природа, профессии, люди

### Пт — ✝️ Вера и добро (ПАССИВНОЕ — девочки гуляют!)
- Сказка / история святого перед сном
- Аудио, короткий ролик
- Задание на выходные (простое, доброе)

### Сб — Первая половина дня (ненадолго)
- Короткое задание или подготовка к воскресному выходу
- Можно творческое, можно домоведение — по настроению

### Вс — 🚶 ВЫХОД ИЗ ДОМА + семейное
- Природа, музей, мастер-класс, парк, театр
- Формат: выход → обед вместе (Вкусная суббота → Вкусное воскресенье)
- Вечер: итоги недели + микро-урок школы родителей (10-15 мин)

## Задания для мамы (Юля)
- Коуч для детей: наблюдения, вопросы, эксперименты
- Свои хобби рядом с детьми: зен-арт, коллажи, готовка, рисование
- НЕ: вышивать, клеить, лепить поделки (не её)

## Задания для папы (Артур)
- Отцовские моменты: что может только папа
- Совместные активности, выходы

## Школа родителей (Юля + Артур)
- Микро-уроки по воскресеньям: 1 принцип / 1 практика / 1 наблюдение
- Источники: Войджитски, Крыгина, Ведяничев
- Воспитание из сильных сторон родителей

## Принципы
- **Без назидания и клише** — через вдохновение
- **Почувствовать и найти в себе** — не навязывать
- **Готовить к старшему возрасту** — трудолюбие, цель, смысл, любопытство
- **Крутые, с которых берут пример** — не «послушные», а вдохновляющие
- **Окружить добротой и любовью** — атмосфера > контроль
- **Все ссылки проверять!** Только рабочие, кликабельные
- **Android** — для рекомендаций игр/приложений

## Банк альтернативных тем (для будущих месяцев)
- **Четыре стихии** — Земля/Вода/Огонь/Воздух (хорошо для лета)
- **Семь дней творения** — библейская структура
- **По ролям** — дочь/подруга/хозяйка/мечтательница (для осени, соц. навыки)
