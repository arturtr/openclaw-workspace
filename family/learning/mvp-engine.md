# 🧬 Движок развития — MVP

> Рабочее название: пока нет. Суть: живая система профориентации и развития через проживание.
> Источник: разговор 06.03.2026. Статус: концепция.

## Чем отличается от Sofia AI и аналогов

| | Sofia AI / аналоги | Наш движок |
|---|---|---|
| Формат | Разовый PDF-отчёт | Живой процесс, растёт с человеком |
| Метод | Западная психометрика | Психометрика + восточные системы + наблюдения |
| Взаимодействие | Заполнил тест → получил файл | Квесты, игры, проживание |
| Обновление | Никогда (снимок) | Постоянно: из игр, наблюдений, обратной связи |
| Для кого | Индивид | Семья (каждый видит себя в контексте других) |
| Глубина | Таланты, роли | Таланты + сезонность + терапия + духовный слой |

## MVP — что строим первым

### Фаза 0: Семья Т. (сейчас)
- **Уже есть:** профили 5 участников, Ба Цзы, натал, наблюдения
- **Уже есть:** 2 сценария (Пробуждение Дома, Картограф)
- **Уже есть:** движок в голове у Кузи (Claude)
- **Нужно:** провести первые игры, собрать логи, увидеть что работает
- **Результат:** проверенная концепция на живой семье

### Фаза 1: Телеграм-бот (1 человек = 1 профиль)
- Бот задаёт вопросы (Big Five + RIASEC + VIA — адаптированные, короткие)
- По дате рождения строит Ба Цзы автоматически
- Генерирует профиль + первую рекомендацию
- **Без квестов** — только диагностика
- Стек: Telegram Bot API + Claude API + Python
- Срок: 2–4 недели на прототип

### Фаза 2: Семейный режим
- Несколько профилей → связи между ними
- Синастрия автоматически
- «Вот как вы дополняете друг друга / вот где конфликты»
- Рекомендации для пары / семьи

### Фаза 3: Квесты
- На основе профилей → генерация персональных сценариев
- Бот как игротехник (асинхронный режим)
- Лог каждой сессии → обновление профиля
- Сезонная адаптация (节气 → тема квеста)

### Фаза 4: Продукт
- Лендинг, оплата, PDF-отчёты (как у Sofia — но с восточными системами)
- Подписка на живую часть (квесты, сезонные рекомендации)
- Для семей, для подростков, для взрослых в переходе
- «Детский лагерь homo divinus» — онлайн-формат

## Тесты — что берём

### Big Five (OCEAN) — открытая версия
- 50 вопросов (IPIP-50)
- Даёт: открытость, добросовестность, экстраверсию, дружелюбие, нейротизм
- Хорошо скрещивается с Ба Цзы (Земля ≈ добросовестность, Вода ≈ нейротизм)

### RIASEC (Holland) — открытая версия
- 42–60 вопросов
- 6 типов карьерных интересов
- Для детей есть адаптации (картинки вместо текста)

### VIA Character Strengths — бесплатный
- 96 вопросов (есть короткая версия 24)
- 24 сильных качества
- Хорошо маппится на добродетели (школа веры!)

### Для детей (6–12)
- Адаптированные вопросы: «что тебе нравится больше — рисовать или считать?»
- Через игру, не через тест
- Или: наблюдение за выборами в квестах → профиль строится неявно

## Ключевое преимущество

**Профиль строится не только из тестов, а из проживания.**

Ребёнок в квесте выбрал зелёную дверь, вырастил дракона, отказался от меча — это данные. Через 10 квестов — паттерн. Через 50 — точный портрет. Без единого теста.

Это то, чего нет ни у кого: **профориентация через приключение**.

## Монетизация (наброски)

1. **Разовый отчёт** (как Sofia): тесты + Ба Цзы + натал → PDF. 3000–5000₽
2. **Семейный пакет:** профили всей семьи + связи. 8000–12000₽
3. **Подписка:** квесты + сезонные рекомендации + обновление профиля. 500–1000₽/мес
4. **Лагерь homo divinus:** групповой онлайн-формат для семей. Сезонный набор

## Конкуренты

- **Sofia AI** — тесты + AI-отчёт. Нет восточных систем, нет живой части
- **Профилум** — школьная профориентация. Массовая, без глубины
- **Барабук, Учи.ру** — edtech, но не профориентация
- **Прямых конкурентов (тесты + Ба Цзы + квесты + сказкотерапия) — нет**
