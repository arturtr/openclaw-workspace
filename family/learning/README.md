## Синхронизация программ → квест

Ежедневная тема маминой школы определяет тему вечернего квеста:

| День | Мамина школа | Квест-тема |
|------|-------------|------------|
| Пн | 📱 Цифровой мир | Волшебные зеркала, границы |
| Вт | 🏠 Домоведение | Замок, хозяйство, забота |
| Ср | 🎨 Творчество | Создание артефакта |
| Чт | 🌍 Мир и люди | Путешествие |
| Пт | ✝️ Вера и добро | Притча-квест |
| Сб | 🚶 Выход | По следам дня |
| Вс | 👨‍👩‍👧‍👧 Семейное | Школа веры → квест |

**Протокол:** Эмбер → тема дня → Кузя → сценарий к вечеру.

## Логи

Каждая игра/урок → запись в `logs/`:
- `logs/quest-YYYY-MM-DD.md` — квесты
- `logs/faith-lesson-NN.md` — школа веры
- `logs/mama-lesson-YYYY-MM-DD.md` — мамина школа

Формат лога:
```
# [Дата] — [Название]
## Участники и роли
## Что было (кратко)
## Реакции детей (наблюдения)
## Что сработало / что нет
## Следующий шаг
```
