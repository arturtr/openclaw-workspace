# Юля — профиль развития

## В семейных сессиях
- **Роль:** со-ведущая, мама, наблюдатель
- **Архетип:** 🌙 Ночная Странница (видит в темноте, слышит то, что молчит)
- **Стиль:** глубина, интуиция, рефлексия. Не экшн — смысл

## Сильные стороны
- Называет невидимое — делает сокрытое видимым
- Проницательность, видит паттерны в людях
- Создаёт пространство одним присутствием
- INFP — глубокие смыслы, аутентичность

## Текущие вызовы
- 七杀 (壬): давление «ты должна соответствовать» — страх выхода
- Выйти из башни, показать свой свет наружу
- Доверие своему методу (не коучинг, не консалтинг — свой путь)
- Баланс: мама + практик + жена

## Как родитель
- **Суперсила:** видит уникальность каждого ребёнка (丁 + INFP)
- **Лучший контакт с детьми:** через образы, сказки, тактильность (♆℞III, ☉♉)
- **MC♋:** материнство = часть призвания, не просто роль
- **♂♑III:** может быть чёткой, дисциплинированной — детям нужна эта рамка
- **Роль:** хранительница смыслов, атмосфера, ежедневные ритуалы
- **Ведёт:** мини-формат в будни (сказка перед сном, вопрос дня), мамина школа

## Точки роста (проигрывать в квестах)
- **Good enough:** «я плохая мать» — ложь от 七杀 → героиня принимает несовершенство
- **Просто побыть:** без повестки → тишина как ресурс, не лень
- **Единый голос с Артуром:** → сцены где два героя решают вместе ДО действия
- **Направить силу наружу:** ♇+Лилит → героиня использует мощь для защиты, не разрушения
