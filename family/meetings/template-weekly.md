# Шаблон еженедельного семейного собрания

> Используй основной шаблон для создания записи о собрании

## Инструкция
1. Скопируй шаблон
2. Создай новый файл с датой (например, `2025-01-27-собрание.md`)
3. Заполни во время собрания
4. Сохрани в этой папке

## Структура собрания
1. Открытие (короткая молитва)
2. Благодарности за неделю
3. Обзор спринта
4. Голос каждого
5. Планирование
6. Решения
7. Молитвенные намерения
8. Закрытие

## Связанные заметки
- Основной шаблон
- Реестр решений
