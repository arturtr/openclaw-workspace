# 🏠 ТЗ на юзабилити жилья — сценарии жизни семьи Трофимовых

> У каждой вещи — своё место. У каждого действия — свой маршрут.
> Этот документ описывает КАК мы живём → и из этого следует ГДЕ что должно быть.
> Используется при выборе жилья, организации пространства, переезде.

## Принципы

1. **Сценарий первичен** — не «куда поставить шкаф», а «что происходит, когда ребёнок приходит из школы»
2. **У каждой вещи — адрес** — если не знаешь куда положить → вещь бездомная → хаос
3. **Маршруты короткие** — от двери до вешалки < 3 шагов. От рюкзака до стола < 5 шагов
4. **Зоны не пересекаются** — зона тишины ≠ зона игр. Зона работы ≠ зона готовки
5. **Дети видят систему** — если правило не очевидно ребёнку → правило не работает

## Сценарии

### 🎒 Ребёнок пришёл из школы

**Маршрут:**
1. Дверь → снять обувь (полка/коврик у входа)
2. Куртку → на свой крючок (подписанный / с эмодзи)
3. Рюкзак → **на своё место рядом с рабочим столом** (не на пол в коридоре!)
4. Руки помыть → ванная
5. Переодеться → домашнее в комнате
6. Перекус → кухня (готовый на столе или в контейнере)
7. Уроки → за свой стол, рюкзак уже рядом

**Что нужно в пространстве:**
- [ ] Крючки по росту ребёнка у входа (2 шт — Маша + Алёна)
- [ ] Полка для рюкзаков рядом с рабочей зоной детей
- [ ] Рабочий стол для каждого ребёнка (или общий, но с разделением)
- [ ] Контейнер для перекуса на кухне (доступный детям)

---

### 🌅 Утро рабочего дня (Артур)

**Маршрут:**
1. Подъём 6:00 → ванная (тихо, не будить семью)
2. Кабинет/уголок → включить комп, заварить чай
3. Работа 6:00–8:00 (тишина)
4. Завтрак с семьёй 8:00
5. Если БЖЖ (Пн/Чт) → собрать сумку → выход

**Что нужно:**
- [ ] Рабочее место изолировано от спальни и детской (дверь или перегородка)
- [ ] Чайный уголок рядом с рабочим местом (чайник, термос, набор)
- [ ] Сумка для БЖЖ — место у входа (кимоно, пояс, полотенце, кусачки)

---

### 🌅 Утро рабочего дня (Юля)

**Маршрут:**
1. Подъём → собрать девочек в школу (6:30-7:25)
2. Завтрак → кухня
3. Проводить детей
4. Коучинг-сессии / блог / керамика / домашние дела

**Что нужно:**
- [ ] Детский утренний маршрут визуализирован (чеклист на стене с эмодзи для Алёны)
- [ ] Школьная форма — готова с вечера (место в детской)
- [ ] Тихое место для онлайн-сессий (коучинг) — дверь закрывается

---

### 🍽️ Ужин и вечер

**Маршрут:**
1. Готовка → кухня (Юля или вместе)
2. Сервировка → дети помогают (тарелки, приборы — в доступном шкафу)
3. Ужин вместе за столом
4. Уборка → дети убирают со стола (чеклист)
5. Вечерний ритуал 20:00 → купание → рюкзак + форма на завтра → зубы → сказка/музыка → отбой 21:00

**Что нужно:**
- [ ] Обеденный стол на 5-6 человек
- [ ] Детская посуда в нижних ящиках (сами достают, сами убирают)
- [ ] Место для «вечернего чеклиста» на стене (визуально, с эмодзи)

---

### 🕯️ Вечер вдвоём (Артур + Юля)

**Маршрут:**
1. Дети в кровати (21:00)
2. Телефоны → на зарядку **в другую комнату** (договор присутствия)
3. Чай вдвоём / разговор / тишина / близость
4. Красный угол → вечерняя молитва (свеча)

**Что нужно:**
- [ ] Место для зарядки телефонов НЕ в спальне
- [ ] Красный угол (икона, свеча) — в спальне или гостиной
- [ ] Чайный уголок доступен вечером

---

### 🏠 Суббота с Ефремом

**Маршрут:**
1. Артур забирает Ефрема утром
2. Активность: Witchcraft Note / Го / прогулка / бар «Вторник»
3. Привозит домой → у Ефрема есть своё место (кровать / диван, полка для вещей)
4. Может играть с Машей (Switch, Го)
5. Артур привозит обратно к 16:00

**Что нужно:**
- [ ] Гостевое спальное место (диван-трансформер или раскладушка)
- [ ] Полка/ящик «Ефрем» — для его вещей, рисунков, скетчбука
- [ ] Ефрем чувствует: у него есть своё место в этом доме

---

### 🍵 Чайная церемония (дома)

**Маршрут:**
1. Подготовка: стол, чабань, набор посуды, чай, вода, термометр
2. Гости приходят → обувь, куртки → место для 4-6 человек за столом
3. Церемония 1-2 часа
4. После — уборка, мойка посуды

**Что нужно:**
- [ ] Место для чайной зоны (отдельный стол или трансформируемое пространство)
- [ ] Хранение чайной посуды (чабань, гайвани, чахай, пиалы)
- [ ] Хранение чая (сухое, тёмное, прохладное)
- [ ] Вода — фильтр или бутилированная

---

### 🧹 Уборка (процессы для детей)

**Маша (8) — может:**
- [ ] Мытьё полов (чеклист в Singularity)
- [ ] Пылесосить
- [ ] Протирать пыль
- [ ] Убирать со стола после еды
- [ ] Загружать/разгружать посудомойку

**Алёна (6) — может:**
- [ ] Убирать игрушки (корзина с понятными отсеками)
- [ ] Вытирать стол после еды (влажная салфетка)
- [ ] Раскладывать чистое бельё по стопкам

**Принцип:** у каждой задачи — чеклист с простыми шагами. Не «убери», а «1. Возьми ведро. 2. Налей воды...»
