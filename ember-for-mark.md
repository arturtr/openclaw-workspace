### 23.03.2026 — СРОЧНО: конвертировать пресет .xmp → .dng для Lightroom Mobile

Юля получила HD-Warm-Golden-Glow.xmp, но не может открыть его на телефоне. **Lightroom Mobile не принимает .xmp напрямую.**

**Что нужно:**
1. Открой пресет в десктопном Lightroom
2. Примени его к любой фотографии
3. Экспортируй эту фотографию как **DNG** (File → Export → Format: DNG)
4. Пришли DNG-файл Юле в Telegram

Юля импортирует DNG в Lightroom Mobile → нажмёт ⋯ → Create Preset → готово.
