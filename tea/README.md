# Чайный бизнес

## Три направления

1. **Обучение** — мастерский курс (7 уроков) + курс для новичков (3 встречи)
2. **Розница** — простая / средняя / премиум линейки
3. **Корпоратив** — чайные подписки и паузы для офисов

## Структура

```
tea/
├── catalog.md              — все сорта с ценами закупки/продажи + формула себестоимости
├── inventory.md            — остатки чая + упаковки
├── sales.md                — сводка продаж (ссылки на заказы)
├── orders/                 — детализация заказов с P&L (по дате)
├── purchases/              — закупки у поставщиков (по дате)
├── clients/                — карточки клиентов (профиль + ссылки на заказы)
├── cards/                  — справочные карточки по сортам (ТКМ, заваривание, цены)
├── suppliers.md            — контакты поставщиков
├── favorites.md            — любимые чаи семьи
├── courses.md              — учебная программа
├── prompts/                — промпты для разбора данных
└── notes/                  — заметки о чае
```

### Потоки данных

```
purchases/ → catalog.md (закупка/г) → orders/ (P&L) → sales.md (сводка)
                                         ↓
                                    inventory.md (остатки)
```

- **Новый заказ:** создать файл в orders/, обновить inventory.md (чай + упаковка)
- **Новая закупка:** создать файл в purchases/, обновить catalog.md и inventory.md
- **Новый клиент:** создать файл в clients/, ссылка на заказ

## Ценообразование

- Базовая наценка: ×2 от закупки
- Себестоимость = (вес × закупка/г) + упаковка
