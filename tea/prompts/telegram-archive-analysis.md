# Промпт: разбор Telegram-архива переписки с клиентом

## Как получить архив

Telegram Desktop -> чат -> ... -> Export chat history -> JSON

## Промпт для Claude Code

```
Разбери Telegram-архив переписки с клиентом чайного бизнеса.

Файл: [путь к result.json]

Задачи:
1. Извлечь все заказы: дата, состав (сорт, вес, цена), итоговая сумма, статус оплаты
2. Собрать профиль клиента: имя, контакты, адрес доставки, предпочтения
3. Выписать ценные идеи/обратную связь от клиента
4. Посчитать: общую сумму, средний чек, частоту заказов, любимые сорта

Формат вывода -- карточка клиента в Markdown:

# Клиент: [Имя]

- Telegram: [id]
- Контакт: [адрес/телефон если есть]

## История заказов

### Заказ N -- [месяц год]
| Сорт | Вес | Цена |
|------|-----|------|
| ... | ... | ... |
| **Итого** | | **X руб.** |

## Сводка
| Показатель | Значение |
|-----------|----------|
| Всего заказов | N |
| Период | дата -- дата |
| Общая сумма | X руб. |
| Средний чек | X руб. |
| Частота | раз в N мес |
| Любимое | сорта |
| Тренд | растёт/стабильно/падает |

## Обратная связь от клиента
- "цитата 1"
- "цитата 2"

Сохранить в: tea/clients/[имя-фамилия].md
```

## Заметки

- Ищи ключевые слова: грамм, рубл, итого, цена, заказ, привез, оплат, перевод
- Отдельно выделяй бизнес-советы от клиента
- Если есть фото -- отметить (Telegram экспортирует фото отдельно)
