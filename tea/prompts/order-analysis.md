# Промпт: разбор заказа у поставщика

## Источники данных

- Email с подтверждением заказа (от mingcha.store или другого поставщика)
- Скриншот заказа
- Текст скопированный с сайта

## Промпт для Claude Code

```
Разбери заказ чая у поставщика.

Данные заказа:
[вставить текст/скрин]

Задачи:
1. Извлечь: номер заказа, дату, поставщика, список позиций
2. Для каждой позиции: название (русское + китайское если есть), тип чая, вес, цена, цена за грамм
3. Посчитать итого и среднюю цену за грамм

Формат вывода:

## Заказ #[номер] -- [дата]

| Поставщик | Сумма | Статус |
|-----------|-------|--------|
| [название] | X руб. | Заказан |

### Позиции

| Сорт | Тип | Вес | Цена | Цена/г |
|------|-----|-----|------|--------|
| Название (иероглифы) | Красный/Улун/... | X г | X руб. | X.XX руб. |

**Итого: X г чая за X руб. (средняя X.XX руб./г)**

Дополнительно:
- Обновить каталог: tea/catalog.md (добавить новые сорта)
- Обновить остатки: tea/inventory.md (статус "В пути")
- Сохранить заказ: tea/purchases/[YYYY-MM-DD]-[поставщик].md
```
