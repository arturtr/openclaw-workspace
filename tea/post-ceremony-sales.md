# Продажа чая после церемоний

**Дата:** 2026-02-03

**Контекст:** Идеи из разговора с Claude про монетизацию чайных церемоний

---

## Проблема

Люди иногда спрашивают про покупку чая, но на церемонии никто не покупает. Причина: люди приходят за опытом, не за покупками. На церемонии они в состоянии погружения, не в режиме "потребитель".

---

## Решение: Follow-up после церемонии

### Схема

```
Церемония → Follow-up в чат (через 1-2 дня) → Заказ в личку → Самовывоз / Яндекс.Доставка
```

### Шаблон follow-up сообщения

```
Вчера пили [НАЗВАНИЕ ЧАЯ].

Кто хочет такой же домой — [ВЕС] за [ЦЕНА СО СКИДКОЙ]₽ (обычно [ОБЫЧНАЯ ЦЕНА]₽).

Скидка до [ДЕНЬ НЕДЕЛИ].

Самовывоз или доставка Яндекс.Доставкой за ваш счёт.

Пишите в личку @artur_trofimov
```

### Пример

```
Вчера пили Те Гуань Инь осенний.

Кто хочет такой же домой — 50г за 500₽ (обычно 600₽).

Скидка до воскресенья.

Самовывоз или доставка Яндекс.Доставкой за ваш счёт.

Пишите в личку @artur_trofimov
```

---

## Почему это работает

| Элемент | Зачем |
|---------|-------|
| Конкретный чай | Память о вчерашнем опыте |
| Цена + скидка | Понятное предложение |
| Дедлайн | Не откладывают |
| Личка | Контролируешь процесс |
| Яндекс.Доставка за их счёт | Не твоя головная боль |

---

## Другие идеи

### "Унести кусочек с собой"
Маленькая порция чая, который пили сегодня (50-100г). Не продажа, а продолжение опыта. Можно предлагать в конце церемонии как опцию.

### Подписка на чай
Раз в месяц присылаешь чай с церемонии + короткую заметку. Для тех, кто не может приходить регулярно. (Реализовать после теста базовых продаж)

---

## Чат участников

Чат может работать как **канал оповещений** (не пытаться строить комьюнити):
- Анонсы (раньше, чем публично)
- Follow-up после церемоний
- Редкие предложения чая

---

## План действий

1. [ ] Сделать заказ у оптовика
2. [ ] Составить простой прайс для себя (чай / грамм / закупка / продажа)
3. [ ] После следующей церемонии — отправить первый follow-up
4. [ ] 3-4 церемонии — собрать данные: какой чай берут, какая цена работает, самовывоз vs доставка
5. [ ] По результатам — решить про каталог/подписку

---

## Принцип

**Не строй магазин — тестируй спрос.**

Сначала 3-4 follow-up, потом выводы и масштабирование.
