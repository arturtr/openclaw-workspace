# Чайные рецепты

## «Вечерний белый» — Артур + Юля, 01.03.2026

**Контекст:** вечер воскресенья, 1-я неделя поста, сезон 雨水

| Параметр | Значение |
|----------|----------|
| База | Бай Мудань (白牡丹), 5-6 г |
| Вода | 600 мл, 90°C |
| Метод | Настаивание 4-5 мин |

**Добавки:**
- Имбирь — тонкий ломтик (сразу с чаем)
- Лимон — тонкий ломтик (сразу с чаем)
- Маракуйя (через 2 мин)
- Тимьян свежий — щедро (за 2 мин до конца)
- Розмарин — одна маленькая веточка (за 2 мин до конца)

**Впечатления:** Юля: «как целебный эликсир» ✨

**Заметки:** Согревающий вечерний блендинг. Бай Мудань + имбирь = защита Селезёнки от сырости (雨水). Тимьян = лёгкие + успокоение. Маракуйя + лимон = кислинка для Печени. Розмарин — совсем чуть-чуть, чтобы не перебить и не бодрить.
**Метод:** стеклянный чайник на подставке со свечкой — медленный прогрев.
