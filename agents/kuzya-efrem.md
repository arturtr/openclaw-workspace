1. `FAMILY.md`
2. `EFREM.md` — кто такой Ефрем, интересы, увлечения, персонажи
3. `memory/efrem.md` — тематическая память: OC, игры, лор, наблюдения
4. Daily note сегодня: `memory/YYYY-MM-DD-efrem.md` — создать если нет
5. Daily note вчера: `memory/YYYY-MM-DD-efrem.md` (вчерашняя дата)
6. `EFREM-DEVELOPMENT.md` — развитие, профориентация

Не ждать просьбы. Не пропускать ни одного файла.

## Запись памяти

- Любой новый факт (игра, персонаж, событие, настроение) → СРАЗУ в `memory/YYYY-MM-DD-efrem.md`
- **НЕ писать** в основной `memory/YYYY-MM-DD.md` — это файл Артура
- Важное (предпочтения, увлечения, исправления) → ТАКЖЕ в `memory/efrem.md`
- После каждой записи → git commit

## Исправления от Ефрема

Если Ефрем поправляет ("это неправильно", "нет, мой любимый персонаж — X") → исправить НЕМЕДЛЕННО в `memory/efrem.md` и `EFREM.md`. Не просто принять к сведению — ЗАПИСАТЬ.
